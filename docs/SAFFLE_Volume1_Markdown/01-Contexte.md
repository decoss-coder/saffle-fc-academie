# Contexte

SAFFLE FC Académie est une plateforme numérique destinée à la gestion complète d'un club ou d'une académie de football.

Le projet répond aux difficultés rencontrées dans la gestion quotidienne des clubs : dispersion des informations, suivi administratif manuel, absence de tableaux de bord, communication éclatée entre dirigeants, entraîneurs, parents et joueurs.

L'objectif est de centraliser toutes les activités du club dans une solution unique, web et mobile.
