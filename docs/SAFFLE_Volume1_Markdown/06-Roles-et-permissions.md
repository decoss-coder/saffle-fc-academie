# Rôles et permissions

Chaque utilisateur possède un rôle déterminant ses droits d'accès.

Les rôles principaux sont :
- Administrateur
- Président
- Comité directeur
- Trésorier
- Coach
- Parent
- Joueur (formation)
- Joueur (Équipe A)
- Responsable communication
- Responsable logistique

Le principe appliqué est celui du moindre privilège.
