# Spécification - Fiche joueur

## Objectif
Centraliser toutes les informations relatives à un joueur.

## Champs obligatoires
- Matricule
- Nom
- Prénoms
- Date de naissance
- Sexe
- Catégorie
- Équipe
- Photo

## Champs complémentaires
- Nom du père
- Nom de la mère
- Tuteur
- Adresse
- Contacts
- Taille
- Poids
- Pied fort
- Poste principal
- Poste secondaire

## Actions
- Créer
- Modifier
- Archiver
- Consulter
- Imprimer
