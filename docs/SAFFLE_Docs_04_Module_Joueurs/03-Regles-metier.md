# Règles métier

FR-JOU-001
Un joueur appartient à une seule catégorie active.

FR-JOU-002
Un joueur appartient à une seule équipe active.

FR-JOU-003
Un joueur de l'Équipe A possède obligatoirement un compte joueur.

FR-JOU-004
Pour les catégories de formation, le parent reste responsable des aspects administratifs.

FR-JOU-005
Un numéro de licence ne peut être attribué qu'à un seul joueur.
