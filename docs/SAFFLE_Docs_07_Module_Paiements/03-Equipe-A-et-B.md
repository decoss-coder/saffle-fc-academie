# Paiements Équipe A et Équipe B

## Objectif

Permettre aux joueurs de l'Équipe A et de l'Équipe B de payer leurs cotisations depuis leur compte personnel.

## Règles fonctionnelles

Pour l'Équipe A et l'Équipe B :

- Le compte joueur est obligatoire.
- Le joueur peut payer lui-même.
- Un parent lié au joueur peut aussi payer.
- Le paiement par parent n'est pas obligatoire.
- Le joueur peut consulter son solde.
- Le joueur peut consulter son historique de paiements.
- Le joueur peut télécharger ses reçus.
- Le joueur ne peut pas modifier un paiement validé.
- Le joueur ne peut pas supprimer un paiement.
- Le joueur ne peut payer que ses propres cotisations.

## Fonctionnalités côté joueur

Le joueur connecté peut :

- voir les cotisations dues ;
- voir les paiements déjà effectués ;
- voir le reste à payer ;
- payer avec Wave ;
- télécharger un reçu ;
- voir le statut d'un paiement ;
- recevoir une notification après validation.

## Fonctionnalités côté parent

Le parent lié au joueur peut :

- voir les cotisations du joueur ;
- payer pour le joueur ;
- télécharger les reçus ;
- consulter l'historique des paiements qu'il a effectués ;
- recevoir une notification après validation.

## Fonctionnalités côté trésorier

Le trésorier peut :

- voir tous les paiements ;
- confirmer un paiement Wave en attente ;
- enregistrer un paiement manuel ;
- annuler un paiement avec motif ;
- générer un reçu ;
- consulter l'audit financier.
