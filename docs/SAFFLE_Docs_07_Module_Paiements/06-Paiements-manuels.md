# Paiements manuels

## Objectif

Permettre au trésorier ou à l'administrateur d'enregistrer les paiements reçus hors ligne.

## Modes concernés

- Espèces
- Virement
- Chèque
- Orange Money
- MTN Money
- Moov Money
- Autre

## Règles fonctionnelles

- Seul un utilisateur autorisé peut enregistrer un paiement manuel.
- Le paiement manuel doit être lié à une cotisation existante.
- Le paiement manuel doit identifier le joueur concerné.
- Le paiement manuel doit indiquer le payeur réel.
- Un reçu doit pouvoir être généré.
- Toute modification doit être journalisée.

## Champs obligatoires

- Joueur concerné
- Cotisation concernée
- Montant payé
- Mode de paiement
- Date du paiement
- Payeur réel
- Observations, si nécessaire
