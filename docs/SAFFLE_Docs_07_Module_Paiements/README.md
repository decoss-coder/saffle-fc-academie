# Module Paiements — SAFFLE FC Académie

Ce dossier contient le cahier des charges fonctionnel du module Paiements.

Le premier besoin prioritaire est la gestion des paiements des joueurs de l'Équipe A et de l'Équipe B, avec possibilité de paiement par :
- le joueur lui-même ;
- un parent ou tuteur lié à sa fiche ;
- le trésorier ou l'administrateur, dans le cas d'un paiement manuel.

La logique retenue est simple : la dette est toujours rattachée au joueur, mais le paiement conserve l'identité réelle du payeur.
