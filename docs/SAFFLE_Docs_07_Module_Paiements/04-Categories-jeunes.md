# Paiements des catégories jeunes

## Objectif

Définir la différence entre les joueurs jeunes et les joueurs Équipe A/B.

## Catégories concernées

- U10
- U12
- U14
- U16
- U18

## Règles fonctionnelles

Pour les catégories de formation :

- Le paiement est principalement effectué par le parent ou tuteur.
- Le joueur peut avoir un compte, mais son accès financier est limité.
- Le joueur peut voir certains éléments si le club l'autorise, mais il ne doit pas payer lui-même par défaut.
- Le parent reste responsable des cotisations, documents et informations administratives.

## Règle de sécurité

Un joueur mineur ne doit pas pouvoir effectuer un paiement sans règle explicite du club.
